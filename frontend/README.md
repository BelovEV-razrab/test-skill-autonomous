# NODECS Landing

Single-page landing for **NODECS — AI Automation Studio** built with pure HTML/CSS.

## Structure

- `frontend/index.html`
- `frontend/styles.css`

## Run

1. Open `frontend/index.html` directly in a browser.
2. Or run a local static server from project root:

```bash
python -m http.server 8000
```

Then open: `http://localhost:8000/frontend/`

## Design Rationale

- Direction: **brutal minimal tech** with hard borders, mono accents, and high-contrast typography.
- Desktop-first layout with large hero hierarchy and wide spacing.
- Responsive behavior collapses grids and process rows for tablet/mobile readability.
- No unnecessary JavaScript: lightweight, semantic, and fast to load.
